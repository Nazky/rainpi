# RainPi©™
## Installation
Clone the repository:
```
git clone https://github.com/Nazky/rainpi.git
```
Go into the directory:
```
cd rainpi
```
If using ARM, run the ARM installer:
```
./install-arm.sh
```
If using ARM64, run the ARM64 installer:
```
./install-arm.sh
```
Just Plug In Your Device and Enter DFU.
Profit.
